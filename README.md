## Getting Started

Clone Repository and install Dependencies

Enter the following into terminal and hit enter

```iterm
git clone https://github.com/daveeS987/socket-client1.git
cd socket-client1
poetry shell
```

Once Shelled up, run the following

`poetry install`  
`python client.py`
